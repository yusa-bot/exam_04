Assignment name  : vbc
Expected files   : *.c *.h
Allowed functions: malloc, calloc, realloc, free, printf, isdigit, write
--------------------------------------------------------------------------------

引数として与えられた数式の計算結果を出力するプログラムを作成してください。
演算子 + * と括弧を正しく処理する必要があります。
式の中で空白文字を処理する必要はありません。
式中の値はすべて0から9までの範囲になります。
予期しない記号や不適切な括弧があった場合、"Unexpected token '%c'\n" と表示し、終了コード 1 で終了します。
（記号が入力の終わりを示す場合は、"Unexpected end of input\n"と出力されます。）
システムコールが失敗した場合は、コード1で終了します。
このディレクトリには、記述する必要のあるコードの冒頭部分があります。

For example this should work:
$> ./vbc '1' | cat -e
1$
$> ./vbc '2+3' | cat -e
5$
$> ./vbc '3*4+5' | cat -e
17$
$> ./vbc '3+4*5' | cat -e
23$
$> ./vbc '(3+4)*5' | cat -e
35$
$> ./vbc '(((((2+2)*2+2)*2+2)*2+2)*2+2)*2' | cat -e
188$
$> ./vbc '1+' | cat -e
Unexpected end of input$
