Assignment name:	picoshell
Expected files:		picoshell.c
Allowed functions:	close, fork, wait, exit, execvp, dup2, pipe
___________________________________________________________________

Write the following function:

int    picoshell(char **cmds[]);

The goal of this function is to execute a pipeline.
この関数の目的は、パイプラインを実行することです。
シェルと同様に、各コマンドを実行し、あるコマンドの出力を次のコマンドの入力に接続する必要があります。

Cmdsには、有効なコマンドのNULL終端リストが含まれています。
各行のコマンドは、execvp の呼び出しに直接使用できる argv 配列です。
各コマンドの最初の引数はコマンド名またはパスであり、execvp の最初の引数として直接渡すことができます。

エラーが発生した場合は、関数は 1 を返さなければなりません
(もちろん、その前に開いているすべてのファイルディスクリプタを閉じる必要があります)。
そうでない場合は、関数はすべての子プロセスを待機し、0を返す必要があります。
このディレクトリには、関数をテストするのに役立つコードを含む main.c というファイルがあります。

Examples:
./picoshell /bin/ls "|" /usr/bin/grep picoshell
picoshell
./picoshell echo 'squalala' "|" cat "|" sed 's/a/b/g'
squblblb
